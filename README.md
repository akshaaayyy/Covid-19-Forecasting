# webmining_project
This is an Machine Learning Project for the course Web Mining in VIT Chennai.

This project provides an insight on the Covid-19 cases in the state of Tamil Nadu, India.

datetime  
requests  
warnings  
plotly.express  
plotly.graph_objects  
seaborn  
scipy.optimize  
sklearn.metrics  
geopandas
